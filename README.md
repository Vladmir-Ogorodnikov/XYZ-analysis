# XYZ-анализ ассортимента

## Описание проекта
В рамках проекта выполнен **XYZ-анализ** ассортимента лекарственных препаратов с целью оценки **стабильности спроса** на основе объема продаж.

XYZ-анализ позволяет разделить товары на группы в зависимости от уровня колебаний спроса и определить, какие позиции:
- продаются стабильно;
- имеют умеренные колебания;
- характеризуются нерегулярным спросом.

Такой подход помогает принимать более обоснованные решения в области:
- управления запасами;
- планирования закупок;
- прогнозирования продаж;
- оптимизации ассортимента.

---

## Цель анализа
Основная цель XYZ-анализа — определить **предсказуемость спроса** для каждого препарата на основе статистических характеристик продаж.

---

## Используемые данные
Для анализа используются следующие признаки:
- **`DR_NDrugs`** — наименование препарата;
- **`DR_Kol`** — объем продаж.

При необходимости из исходного датасета отбираются только препараты, включенные в анализ.

---

## Методология анализа

### 1. Подготовка данных
На первом этапе данные очищаются и приводятся к удобному для анализа формату:
- сбрасывается индекс датафрейма;
- отбираются необходимые столбцы;
- фильтруются препараты, включенные в выборку для анализа.

### 2. Расчет коэффициента вариации
Для каждого препарата рассчитывается **коэффициент вариации**.

Формула расчета:

`CV = std / mean`

где:
- `std` — стандартное отклонение объема продаж;
- `mean` — средний объем продаж.

Коэффициент вариации показывает, насколько сильно объем продаж отклоняется от среднего значения.
Чем меньше значение `CV`, тем стабильнее спрос на препарат.

### 3. Классификация по группам XYZ
После расчета коэффициента вариации препараты распределяются по трем категориям:

- **X** — стабильный спрос, низкий уровень колебаний  
  `cv < 0.10`

- **Y** — умеренные колебания спроса  
  `0.10 <= cv < 0.25`

- **Z** — нестабильный спрос, высокая изменчивость  
  `cv >= 0.25`

---

## Интерпретация результатов

### Группа X
Препараты с наиболее стабильным спросом.  
Для таких позиций можно:
- использовать стандартные методы прогнозирования;
- поддерживать регулярные поставки;
- планировать запасы с высокой степенью уверенности.

### Группа Y
Препараты с умеренной вариативностью спроса.  
Для них рекомендуется:
- учитывать возможные сезонные колебания;
- чаще пересматривать прогнозы;
- применять более гибкое планирование закупок.

### Группа Z
Препараты с нестабильным спросом.  
Для таких позиций целесообразно:
- минимизировать избыточные запасы;
- проводить дополнительный анализ причин колебаний;
- использовать осторожный подход к закупкам и хранению.
